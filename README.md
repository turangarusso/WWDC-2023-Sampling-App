## Key Features

* Explain the A/D Confersion
  - Show the difference between Analog/Digital Signal .
* TC Signal Vs TD Signal
* Show canonical signal Sampling with a custom Plot
  - Sin Signal.
  - Rect Signal.
  - Triangular Signal.
* Show different sampling rate effects
* Microphone showcase.
